在二十进制中，我们除了使用数字0-9以外，还使用字母a-j（表示10-19），给定两个二十进制整数，求它们的和。 输入是两个二十进制整数，且都大于0，不超过100位； 输出是它们的和（二十进制），且不包含首0。我们用字符串来表示二十进制整数。
```
/*
二十进制相加
sample
input

1234567890+abcdefghij
99999jjjjj+9999900001

output

bdfi02467j
iiiij00000
*/
```
